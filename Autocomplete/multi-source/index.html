Multi-source Autocomplete

This autocomplete display items from multiple sources. It includes:

  • pre-defined items, i.e., links to our Autocomplete Documentation and Github respository
  • recent searches made by the user
  • suggested searches powered by Algolia's Query Suggestions feature

You'll only see recent searches appear once you've submitted (pressed enter) on a query.

To see results, hit enter once you've selected a suggested or recent search or typed a query.
